# Source Tree

`src/` — это единственный источник deployable исходников.

Сюда не складываются:

- временные заметки;
- архив задач;
- traceability;
- тестовые отчеты;
- ad-hoc выгрузки для обсуждения.

Подкаталоги:

- `cf/` — исходники основной конфигурации; этот каталог должен оставаться чистым source-of-truth для `LoadConfigFromFiles`, статического анализа BSL и сравнения изменений в Git
- `cfe/` — исходники расширений
- `epf/` — внешние обработки
- `erf/` — внешние отчеты

Внутрь deployable `src/cf` не кладутся agent-routing или descriptive markdown-файлы вроде `AGENTS.md` и `README.md`.
