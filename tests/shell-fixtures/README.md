# Shell Fixtures

Сюда помещаются deterministic shell-level проверки для launcher-скриптов, profile loading и artifact contracts.

Типовые кейсы:

- runtime profile parsing;
- adapter command resolution;
- `summary.json` / `stdout.log` / `stderr.log` contract;
- диагностическая классификация ошибок без реальной 1С-платформы.
