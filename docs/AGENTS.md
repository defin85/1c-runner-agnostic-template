# Docs Routing

Этот каталог является shared template-managed docs layer.

- В template source repo authoritative onboarding начинается с [docs/agent/index.md](agent/index.md).
- В generated repo authoritative onboarding начинается с [docs/agent/generated-project-index.md](agent/generated-project-index.md), затем продолжается в `automation/context/project-map.md`.
- Template maintenance остаётся отдельным маршрутом в [docs/template-maintenance.md](template-maintenance.md) и не должен подменять feature-delivery onboarding.
- Durable docs в `docs/` маршрутизируют, но не заменяют project-owned domain truth generated repo.
